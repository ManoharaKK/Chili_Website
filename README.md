# Chili Website

A Next.js website for Sri Lankan spices.

## Quick Start

### ⚡ Faster Clone Recommended

This repository contains large media files (~26 MB). For a **much faster clone**, use:

```bash
git clone --depth 1 https://github.com/chamika-damith/Chili_Website.git
cd Chili_Website
```

The `--depth 1` flag downloads only the latest commit, making cloning significantly faster.

### Standard Clone

```bash
git clone https://github.com/chamika-damith/Chili_Website.git
cd Chili_Website
```

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Repository Size Notice ⚠️
